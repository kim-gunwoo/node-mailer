# 노드 메일 예제

## NODE 폴더

```bash
# node
npm i express node-mailer

#nodemon
npm i -D nodemon

```

## Build Setup

```bash
# install dependencies
npm install

# smtp 서버 설정
dev_template.js 파일 dev.js 로 수정후 내용 입력

# serve with hot reload at localhost:8080
npm run dev

```
